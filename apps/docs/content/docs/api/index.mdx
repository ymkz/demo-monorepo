# APIリファレンス

## 概要

Spring Bootで構築されたREST APIの仕様です。

## ベースURL

```
http://localhost:8080
```

## 認証

（認証方式が決定次第追記）

## エンドポイント一覧

APIエンドポイントの詳細は、OpenAPI仕様から自動生成されます。

## OpenAPI仕様

生のOpenAPI JSONは以下から取得できます：

```bash
# APIサーバー起動時
curl http://localhost:8080/v3/api-docs

# またはビルド時に生成されたファイル
cat apps/api/build/openapi.json
```
