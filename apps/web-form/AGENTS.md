## CONVENTIONS

### TanStack Start Patterns
- Routes are file-based under `src/routes/`
- Root document lives in `src/routes/__root.tsx`
- Router configuration lives in `src/router.tsx`
- `src/routeTree.gen.ts` is generated by TanStack Router and must not be edited manually

### API Routes
- Prefer backend OpenAPI generated client for API access
- Use TanStack Start server functions only for web-form specific server behavior
- Environment variables for configuration

### Development
- Port 3000 (via `vite dev`)
- Vite for fast HMR
- Biome for linting (run `pnpm lint`)

## ANTI-PATTERNS

| Pattern | Why Forbidden |
|---------|---------------|
| Manual API types | Use generated OpenAPI types instead |
| Hardcoded secrets | Use environment variables |
| Editing routeTree.gen.ts | Generated by TanStack Router |
